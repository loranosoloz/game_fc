FM24 2013/2014 Retro Database

Want to relieve the last 10 years of football and rewrite history? Now you can with the FM24 2013/14 Retro Database created by MrTini23.

FMInside Dec 9, 2023 Last modified: 9 Dec 48,720 views 8,561 downloads
Data updates FM24
Direct download

Everything in this FM24 database retro database has been reverted to 2013 using data from Football Manager 2014. The database includes the likes of Messi at Barcelona, Ronaldo at Real Madrid along with wonderkids from back in the day like Wellington Nem. With the support staff EDT file you will also see 14,500 regens appear over 10 years including the likes of Erling Haaland, Kylian Mbappe and more.

Still unsure? Check out this video by FM Creator Lollujo which explores the mod in detail.

How to play Season 2013/14 on FM24

FMF version

Install the support staff file in the following folder

STEAM

C:\Program Files (x86)\Steam\steamapps\common\Football Manager 2024\data\database\db\2400\edt\permanent

EPIC GAMES

/Users/Shared/Epic Games/FootballManager2023/database/data/db/2400/edt/permanent

MAC

Library/Application Support/Steam/steamapps/common/Football Manager 2024/database/data/db/2400/edt/permanent

The database FMF should be installed in the following location:

C:\Users\xxxxx\Documents\Sports Interactive\Football Manager 2024\editor data
